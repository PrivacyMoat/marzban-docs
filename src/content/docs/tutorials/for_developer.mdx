## Установка библиотеки

Для начала работы с Marzban API выполните установку последней версии:

```bash
pip install marzban
```

## Быстрый старт
<Steps>
1. Импортируйте библиотеку и настройте клиент:
    ```python
    from marzban import MarzbanAPIClient

    client = MarzbanAPIClient(base_url="http://your-marzban-url", token="your_api_token")
    ```
2. Получите список пользователей:
    ```python
    users = await client.get_users()
    print(users)
    ```
3. Добавьте нового пользователя:
    ```python
    user_data = {
        "username": "new_user",
        "password": "secure_password"
    }
    new_user = await client.create_user(user_data)
    print(new_user)
    ```
4. Управляйте узлами:
    ```python
    nodes = await client.get_nodes()
    print(nodes)
    ```
</Steps>
